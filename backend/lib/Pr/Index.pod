=head1 NAME

Bwtr::Index - BWTool Report Viewer

=head1 ADMIN DOCUMENTAION

Bwtr is a plugin based system. Reporters are implemented by sub-classing
L<Bwtr::Reporter>.

=head1 TECH DOCUMENTATION

The Bwtr backend is written in perl, using the the L<Mojolicious>. See
L<Mojolicious::Guides> for details.

=over

=item *

L<Bwtr> - the mojo application starting point

=item * 

L<Bwtr::Config> - the configuration object

=item * 

L<Bwtr::Cfg> - the configuration Syntax

=item * 

L<Bwtr::RpcService> - the json rpc api

=item *

L<Bwtr::Reporter> - the report base class

=back

=cut
